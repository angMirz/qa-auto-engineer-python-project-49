## Instructions

### Setup

```bash
make install
```

### Build and package-install

```bash
make build
```
```bash
make package-install
```

## Run games

Game: "Evenness Check"
```bash
brain-even
```
Game: "Calculator"
```bash
brain-calc
```
Game: "NOD"
```bash
brain-gcd
```
Game: "Arithmetic progression"
